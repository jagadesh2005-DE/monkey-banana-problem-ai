# Monkey Banana Problem (Artificial Intelligence)

This repository contains a Python implementation of the **Monkey Banana Problem**, a classic Artificial Intelligence problem used to demonstrate problem-solving using states and actions.

## Problem Description

In this problem:

* A monkey is in a room.
* A banana is hanging from the ceiling.
* A box is present in the room.

The monkey must perform the following actions to get the banana:

1. Move to the box
2. Push the box to the banana's position
3. Climb onto the box
4. Grab the banana

The goal is for the monkey to **successfully obtain the banana**.

## Requirements

* Python 3.x

## How to Run

Run the program using Python:

```
python monkey_banana_problem.py
```

## Example Input

```
Enter monkey position: A
Enter box position: B
Enter banana position: C
```

## Example Output

```
Monkey moves from A to B
Monkey pushes box from B to C
Monkey climbs onto the box
Monkey grabs the banana

Goal Achieved: Monkey got the banana!
```
